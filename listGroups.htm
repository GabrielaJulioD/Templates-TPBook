List Groups

With this function you will be able to add to your page a Group List easy to adapt to your project since its presentation has different ways of classification such as: Ordered, unordered, with lists within lists, among others. This function is very attractive because of the adaptability and organization it provides. 

                
       
 <!-- Ordered list -->

       <div class="contentainer-list" style="display: flex; justify-content: space-around; font-size:medium">
       <div class="order-lists" style="font-family: 'Noto Sans', sans-serif;
       font-size: 17.5pt;"> <h2>Ordered list</h2>
       <ol type="i" start="1">
       
<!-- Change the list items inside the li tag -->

                            <li> Elemento 1</li>
                            <li> Elemento 2</li>
                            <li> Elemento 3</li>
                            <li> Elemento 4</li>
                            <li> Elemento 5</li>
                        </ol>
                    </div>
       
 <!-- Unordered list -->

                    <div class="unorder-list" style="font-family: 'Noto Sans', sans-serif;
                   font-size: 17.5pt;"><h2>Unordered list</h2>
                        <ul type="square">
       
<!-- Change the list items inside the li tag -->

                            <li> Elemento 1</li>
                            <li> Elemento 2</li>
                            <li> Elemento 3</li>
                            <li> Elemento 4</li>
                            <li> Elemento 5</li>
                        </ul>
                    </div>
     
   <!-- List in list -->

       
<!-- Change the list items inside the li tag -->

                    <div class="list-in-list" style="font-family: 'Noto Sans', sans-serif;
                   font-size: 17.5pt;"><h2>List in list</h2>
                        <ol type="A" start="3">
                            <li>Element 1 
                                <ul type="disc">
                                    <li> Subelement 1</li>
                                    <li> Subelement 2</li>
                                    <li> Subelement 3</li>
                                </ul>
                            </li>
                            <li> Element 2
                                <ol type="a" start="7">
                                    <li> Subelement 4</li>
                                    <li> Subelement 5</li>
                                    <li> Subelement 6</li>
                                </ol>
                            </li>
                            <li>Element 3</li>
                        </ol>
                    </div>
                 </div>
