Jumbotron

The jumbotron is a tool that will allow you to enter page headers with useful information for users. 

                
                    <!-- Jumbotron container -->
                    <div
                    id="basic-jumbotron"
                    style="
                      font-family: 'Noto Sans', sans-serif;
                      background-color: #f0f0f0;
                      border-radius: 4px;
                      margin: 2%;
                      padding: 20px;
                      box-shadow: 0 2px 2px rgba(0, 0, 0, 0.2);
                      color: #1c1c11;
                    "
                  >
<!-- jumbotron title -->

<!-- Change the title of the Jumbotron inside the h1 tag -->

<h1>Jumbotron title</h1>
