Checkbox

It is an input element that allows you to insert a vector or array of values. The value attribute is used to define the value sent by the checkbox. The checked attribute is used to indicate that the element is selected. 

                

<!-- checkbox conainer -->

                    <div>
<!-- checkbox label -->

 <!-- Change the text inside the label tag -->

                    <label for="comunicacion"> Checkbox</label><br>

 <!-- Change the text inside the input tag -->

<!-- checkbox field 1 -->
                    <input type="checkbox" name="" id="option1">Checkbox option 1<br>
<!-- checkbox field 2 -->
                    <input type="checkbox" name="" id="option2">Checkbox option 2<br>
<!-- checkbox field 3 -->
                    <input type="checkbox" name="" id="option3">Checkbox option 3<br>
                </div>
